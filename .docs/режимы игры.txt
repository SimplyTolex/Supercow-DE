два типа игры:

- прохождение
  - начинаешь с 1-1 и проходишь все уровни по порядку
  - для игры можно выбрать только текущий и предыдущие уровени (при этом счёт будет восстановлен до того, который был на момент выбранного уровня)

- свободная игра
  - можно выбрать любой открытый уровень
    - чтобы открыть уровень, его нужно сперва пройти в прохождении
  - счёт ведётся только для выбранного уровня. общий счёт там только для красоты, он нигде не используется

два вида кампаний:

- оригинальная
  - воссоздание оригинальных уровней супер коровы

- своя
  - новые, более сложные уровни. Что-то на подобие "новой игры+"
